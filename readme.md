# **About UIXBase/Laravel Components**
UIXBase is a Laravel components framework builder. 
We develop this package base on `Laravel/framework` and `TailwindCss` for styling.

## Installation
### Requirements
```bash
composer require uixbase/uixbase-laravel
```
